# [DPR-003] 개인정보 처리 안내 문구 작성

## 1. 목적
- 사용자가 진단을 제출하기 전 체크해야 하는 '개인정보 수집 및 이용 동의' 문구를 구체화하고 UI에 반영한다.

## 2. 구현 범위
- 진단 페이지의 동의 체크박스 문안 및 상세 보기 모달/페이지

## 3. 처리 로직
1. 수집 항목 명시 (이름, 연락처, 진단 답변)
2. 이용 목적 명시 (전문가 상담 안내, 진단 리포트 생성 및 전송)
3. 보유 기간 명시 (예: 파기 요청 시까지 또는 수집 후 1년 등)
4. UI 적용 (Q-002 연동)

## 4. 완료 기준
- [ ] 동의 문구가 컴포넌트에 반영됨
- [ ] 체크박스 UI 동작 및 유효성 검증

## 5. 관련 태스크
- 선행: DB-011
- 후행: DPR-001, Q-002
