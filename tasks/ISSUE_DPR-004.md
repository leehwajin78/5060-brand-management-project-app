# [DPR-004] AI 분석 동의 문구 작성

## 1. 목적
- 제출된 진단 내용이 AI에 의해 처리됨을 투명하게 안내하고 동의를 얻는 UI를 구성한다.

## 2. 구현 범위
- 진단 폼 하단 'AI 처리 동의' 항목

## 3. 처리 로직
1. "입력하신 진단 답변은 AI 분석을 위해 사용됩니다. 이름과 연락처는 전송되지 않습니다." 등의 문안 작성
2. AI 응답은 100% 완벽하지 않을 수 있으며, 담당자가 최종 검수한다는 내용을 명시
3. DPR-003(개인정보 동의)과 통합된 체크박스로 관리하거나 별도 체크박스로 분리 적용 (정책에 따름)

## 4. 완료 기준
- [ ] AI 분석 동의 문구가 UI 컴포넌트에 반영됨

## 5. 관련 태스크
- 선행: DPR-002
- 후행: Q-002, FE-001
