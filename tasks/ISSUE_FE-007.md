# [FE-007] 관리자 인증 — 환경변수 패스워드 체크 (미들웨어/가드)

## 1. 목적
- 인가되지 않은 사용자가 관리자 콘솔(`/admin`)에 접근하는 것을 차단하기 위해 간단하지만 확실한 인증 흐름을 구축한다. 

## 2. 구현 범위
- `/admin` 하위 경로에 대한 접근 제어(Next.js Middleware 활용)
- 패스워드 입력 폼 (로그인 화면) 구현
- 세션(쿠키)을 통한 로그인 유지

## 3. 입력/출력
### Input
- 관리자 패스워드 텍스트 입력

### Output
- 인증 성공 시 인증 쿠키 발급 및 리다이렉트
- 인증 실패 시 에러 메시지 표시

## 4. 처리 로직
1. `middleware.ts`를 생성하여 `/admin` 경로로 접근 시 인증 쿠키 검사 로직 추가.
2. 인증되지 않은 경우 `/admin/login` 화면으로 리다이렉트.
3. 로그인 화면에서 패스워드를 입력받아, 서버의 환경변수(`ADMIN_PASSWORD`)와 비교 (Server Action 사용 권장).
4. 패스워드 일치 시 HttpOnly/Secure 설정된 세션 쿠키 발급.
5. 로그인 폼 클라이언트 측 검증(빈 값 제출 방지) 및 UI 피드백 구현.

## 5. 완료 기준
- [ ] 미인증 상태로 `/admin` 접속 시 로그인 페이지로 리다이렉트 되는지 확인
- [ ] 올바른 패스워드 입력 시에만 목록 페이지로 진입 가능한지 확인
- [ ] 인증 세션(쿠키)이 정상적으로 동작하는지 확인

## 6. 관련 태스크
