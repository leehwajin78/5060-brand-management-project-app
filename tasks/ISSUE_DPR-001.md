# [DPR-001] 고객 동의 체크박스 UI (개인정보 수집·AI 처리 동의)

## 1. 목적
- 개인정보보호법 및 AI 서비스 제공 약관 준수를 위해 진단 제출 전 사용자의 명시적인 동의를 얻는 UI를 제공한다.

## 2. 구현 범위
- 진단 폼 하단 (제출 버튼 위)에 체크박스 및 안내 문구 추가
- 개인정보 수집 및 이용 동의 (필수)
- AI 분석을 위한 데이터 처리 동의 (필수)

## 3. 입력/출력
### Input
- 사용자 체크박스 인터랙션 (boolean)

### Output
- 폼 제출 시 동의 여부 상태값 검증 (미동의 시 제출 차단)

## 4. 처리 로직
1. 16문항 진단 폼 컴포넌트 하단에 약관 동의 체크박스 추가.
2. 각 약관의 세부 내용을 볼 수 있는 간단한 모달이나 링크 제공 (DPR-003, DPR-004 연동).
3. 체크박스가 모두 선택되어야만 "진단 제출" 버튼이 활성화되도록 유효성 검사 적용.

## 5. 완료 기준
- [ ] 약관 동의 체크박스 2개(혹은 통합 1개) UI 렌더링 확인
- [ ] 필수 동의 항목 미체크 시 폼 제출이 차단되고 안내 메시지가 표시되는지 확인

## 6. 관련 태스크
- 선행: Q-002 (진단 폼 UI)
- 후행: FE-001 (진단 폼 클라이언트 유효성 검증)
